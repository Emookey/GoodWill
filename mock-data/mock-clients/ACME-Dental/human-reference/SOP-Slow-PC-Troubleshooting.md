# SOP - Slow PC Troubleshooting

## Purpose

Use this SOP when a user reports that a workstation is slow, freezing, or taking a long time to open applications.

## Initial Questions

- Is the issue affecting one workstation or multiple users?
- Is the slowness limited to one application?
- Did the issue begin after updates, password change, or software install?
- Is the workstation using HDD storage or SSD storage?
- How much RAM is installed?
- Is disk usage stuck near 100 percent?

## Basic Troubleshooting Order

1. Check Task Manager for CPU, memory, disk, and startup impact.
2. Check whether the workstation uses HDD storage.
3. Check available disk space.
4. Check Windows Update status.
5. Check for pending reboot.
6. Check disk health if the device uses an HDD.
7. Confirm whether the application is local or network-hosted.
8. Compare with another user's workstation if a shared application is involved.
9. Recommend SSD/RAM upgrade or replacement when specs support that recommendation.

## Important Notes

- A workstation with 4GB RAM and an HDD should be considered under-spec for normal Windows 10 business use.
- A workstation with 8GB RAM and an HDD may have application performance problems, especially with QuickBooks or large Office files.
- Stronger hardware does not rule out app-specific issues, driver issues, or profile corruption.
