# Roadmap

## Current phase

- Validate GTX 1060 6GB stability.
- Replace/verify CMOS battery.
- Continue local model testing.
- Prove small-image vision workflows.
- Compare Odysseus and Open WebUI vision behavior.
- Test long-standing, multi-turn conversations.
- Keep all testing on mock data.

## Next phase

- Build mock MSP knowledge documents.
- Test helpdesk triage prompts.
- Test SaaS alert interpretation prompts.
- Test screenshot-to-ticket workflows.
- Document good prompts and failure cases.
- Improve backup and restore routines.

## Later phase

- Consider storage expansion.
- Consider stronger hardware if the workflow proves useful.
- Evaluate stricter access controls.
- Define read-only client-data access rules before any real workplace usage.

## Not in scope yet

- Real client data ingestion
- Autonomous remediation
- Public service exposure
- Production deployment
- Larger data-center GPUs without a proper power/cooling plan
